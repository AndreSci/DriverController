[GET] /PulseDeviceByReader
    Принимает FID считывателя.
    Отправляет команду пульс на устройство связанное со считывателем.

[GET] /PulseByCameraID
    Принимает FID камеры.
    Отправляет команду пульс на устройство связанное со считывателем.

[GET] /StateByCameraID
    Принимает FID камеры.
    Возвращает статус устройства из кэша 'Наблюдателя' в виде json.

[GET] /ManualControlDevice
    Принимает FID или host:port устройства плюс строковый байт-код.
    Отправляет запрос напрямую в устройство и возвращает результат запроса.

[POST] /ActionAddCards  (В ТЕСТЕ)
    Принимает host:port устройства или ничего (где ничего флаг для записи во все устройства из БД) и список карт GUID.
    Циклично рассылает карты по всем устройствам и возвращает ответ с результатами о выполнении записи всех устройств
